# test-datasub

## Installation
`npm install`

## Task 1

To run task 1 `node task1.js`
or
Compile typescript `npx tsc task1.ts` and run `node task1.js`

To run task 2 `node task2.js`
or
Compile typescript `npx tsc task2.ts` and run `node task2.js`
